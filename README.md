### OBS: You need to have the gem *bundler* installed.

# BASIC SETTINGS
## to install gems

Run

```shell
bundle install
```


# TO RUN TESTS:
## to execute the tests

```shell
rspec
```


# TO RUN THE APP:
# to run the app in the rack default port (9292)

```shell
rackup
```

Then, now you can access the app in *localhost:9292*.
